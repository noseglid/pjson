Introduction to pjson

The pjson library is intended to be a fast and easy to use JSON interpreter. It turns any valid JSON string into various container types as std:string, int, std::vector and std::map. Mainly two functions are the base of the interface; Json::serialize and Json::deserialize.

This library follows the JSON definition in RFC4627.

Quick start

Here is a quick example to get you started - more detailed tutorials are below. 

#include <pjson/pjson.hpp>

#include <string>
#include <iostream>

int main(void)
{
        /* A JSON string. */
        std::string strJSON = "{\"action\" : \"update\", \"itemid\" : 38, \"cost\" : 12.99}";

        /* Parse the JSON string */
        Json::Value jv = Json::deserialize(strJSON);

        /* We can now use 'jv' to retrieve the values. */
        std::cout << "Action: " << jv["action"].asString() << std::endl;
        std::cout << "  Item: " << jv["itemid"].asInt()    << std::endl;
        std::cout << "  Cost: " << jv["cost"].asNumber()   << std::endl;

        return 0;
}

Action: update
  Item: 38
  Cost: 12.99
