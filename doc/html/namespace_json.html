Serializes an object. This will give the json string representation of any object you give it. You can give almost any type to this funtion - including but not limited to:

  • int, float, double
  • std::string
  • map<std::string, T>
  • vector<T>
  • bool
  • const char*
Parameters:
objectThe object to serialize
formatHow to format the string
Exceptions:
Json::Exceptionif any error is encountered.
Returns:
String representation of the object

Variable Documentation

Json::Value Json::deserialize

Deserializes a JSON string and returns an instance of Json::Value. Json::Value provides an easy to use interface to read data.

A JSON string can be anything which is described in RFC4627. For example 

 { "timestamp" : 1319745798, "mood" : "happy" }
Parameters:
jsonThe json object as a string.
Exceptions:
Json::ExceptionIf the string can not be interpreted as JSON.
Returns:
A representation of the JSON object.
Examples:
doc-array.cpp, doc-basic.cpp, and doc-object.cpp.
